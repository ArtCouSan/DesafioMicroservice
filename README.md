# Microservice

Spring Boot + Microservice + Swagger2

# Necessario

- JDK 11
- MySQL
- Maven

# Docs

- http://localhost:8080/gateway/order/swagger-ui.html#
- http://localhost:8080/gateway/product/swagger-ui.html#
- http://localhost:8080/gateway/user/swagger-ui.html#
